# Perf History

This folder is for local performance-run artifacts generated by
`scripts/run_perf_harness.py`.

Generated files stay local and are intentionally ignored by git so we can keep
an ongoing regression trail without turning the repo into an artifact dump.

The harness maintains:

- `latest.json` and `latest.md` for the newest recorded run
- `history.csv` for quick spreadsheet-style comparisons over time
- `history.jsonl` for append-only machine-readable history
- `runs/` for timestamped per-run JSON and Markdown snapshots

Typical command:

```bash
python scripts/run_perf_harness.py --base-url http://127.0.0.1:8080 --iterations 3 --format markdown --label baseline
```
