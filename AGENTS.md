# Agent Instructions

`CONTRIBUTING.md` is the canonical contributor and agent rails document for this repository.

Before editing code, tests, docs, workflows, or release artifacts:

1. Read `CONTRIBUTING.md`.
2. Follow its safety boundaries for secrets, live data, logs, history databases, and local-only config.
3. Use the validation tier that matches the change risk.
4. Keep public issues, PRs, and docs reproducible from sanitized/public repo state.

Do not copy raw logs, history database rows, secret config, SSH material, private topology, or maintainer-local data into prompts, issues, commits, or handoffs unless the maintainer explicitly approves that exact source and purpose.
